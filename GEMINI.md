# Project Context: AWS AI Fullstack Developer Portfolio

This project is a comprehensive portfolio building exercise structured into 31 sprints (0-30). Each sprint is managed in its own uniquely named branch.

## Core Mandates
- **Branch Strategy:** Work strictly within the current sprint branch. Do not merge into `main` without explicit instruction.
- **Sprint Alignment:** Ensure all features implemented align with the sprint definitions in `PLAN.md`.
- **AWS Integration:** Focus on professional-grade AWS service implementations (Bedrock, Lambda, Amplify, etc.) as the project progresses.

## Coding Standards
- **Framework:** Next.js (App Router) for the frontend.
- **Styling:** Tailwind CSS for a modern, responsive UI.
- **Backend:** AWS Serverless architecture.
- **Documentation:** Maintain clear comments and update README/PLAN as features are added.

## Sprint Roadmap (Reference)
*See `PLAN.md` for the full 31-sprint list.*
- **Current Phase:** Sprint 0 (`00_where_am_i`) - Project initialization and setup.

## Technical Preferences
- Prefer TypeScript for type safety.
- Use modular components for UI.
- Follow AWS best practices for security and scalability.
