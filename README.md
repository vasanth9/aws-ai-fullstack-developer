# 120-Day AWS AI Full Stack Masterclass

This repository tracks a 120-day journey from serverless fundamentals to autonomous industrial AI systems.

## 🚀 The Roadmap
- **Phase 1-6 (90 Days):** 30 Advanced Sprints covering Global APIs, Event-Driven Architectures, and Agentic RAG.
- **Phase 7 (30 Days):** 3 Industrial AI Capstones (Retail Intelligence, OmniMedia Studio, and Global Cyber-Guard).

## 🛠 Project Dashboard
A Next.js 15+ dashboard is included to track progress and technical milestones.

### Launch the Dashboard
```bash
cd dashboard
npm run dev
```

### Curriculum Structure
- **Sprints 1-5:** Distributed Serverless Architectures
- **Sprints 6-10:** Intelligent Media & Vision
- **Sprints 11-15:** High-Scale Data Streams
- **Sprints 16-20:** Agentic AI & RAG
- **Sprints 21-25:** Cloud-Native SaaS
- **Sprints 26-30:** Autonomous Ops
- **Days 91-120:** Industrial Capstones

## 📐 Technical Requirements
- **Observability:** CloudWatch + X-Ray.
- **Security:** Zero-Trust IAM + KMS.
- **Infrastructure:** AWS CDK (TypeScript).
- **Frontend:** Next.js (App Router) + Tailwind CSS.
