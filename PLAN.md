## Technical Requirements for Each Project:
1.  **Observability:** Every project must have CloudWatch Dashboards and X-Ray tracing.
2.  **Security:** IAM Least Privilege, KMS Encryption, and WAF protection where applicable.
3.  **CI/CD:** Automated deployments via GitHub Actions or AWS CodePipeline.
4.  **AI Refinement:** Use AI not just for the features, but for writing unit tests and optimizing SQL/NoSQL queries.
